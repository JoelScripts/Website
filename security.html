Data Security & Hosting Information

This page provides information about how your data is secured and where your information is stored. This is important for GDPR compliance and user trust.

1. Hosting Provider Information

Flying With Joel's website is hosted on a third-party hosting provider. When selecting our hosting provider, we ensure:

  • GDPR compliance is certified or contractually guaranteed
  • Data centers are located in compliant jurisdictions
  • Data processing agreements are in place
  • Security certifications (ISO 27001 or equivalent) are maintained
  • Regular security audits are conducted

2. Data Security Measures

2.1 Transport Security

  • HTTPS/SSL Encryption: All communication between your browser and our website is encrypted using SSL/TLS protocols
  • Certificate Verification: Our SSL certificate is issued by a trusted Certificate Authority
  • Secure Headers: HTTP security headers are implemented to prevent common attacks

2.2 Data Storage

  • Client-Side Storage: Timezone preferences and cookie consent choices are stored locally on your device using browser localStorage (NOT sent to our servers)
  • Contact Form Data: Email enquiries are stored securely with encrypted backups
  • No Unnecessary Data: We do not collect or store behavioral data (page views, time spent, etc.)

2.3 Access Controls

  • Limited personnel have access to stored data
  • Access is restricted to business-critical functions only
  • Unique credentials are required for administrative access
  • Access logs are maintained for audit purposes

3. Where Your Data Is Stored

3.1 Your Device (Preferred)

  • Timezone Preference: Stored in your browser's localStorage
  • Cookie Consent: Stored in your browser's localStorage
  • Storage Duration: Until you clear your browser data or withdraw consent
  • Security: Only accessible by this website; never transmitted to our servers

3.2 Our Hosting Provider's Servers

  • Email Enquiries: Contact form submissions are stored securely on our hosting provider's servers
  • Website Files: HTML, CSS, JavaScript, and static assets
  • Location: Data centers in GDPR-compliant regions
  • Retention: Contact data retained for 2 years then securely deleted

3.3 Third-Party Services

  • Google Fonts: Font files cached by Google; limited data only
  • Twitch: Embedded streams may collect viewing data per Twitch's privacy policy
  • Your Responsibility: Review each third-party's privacy policy for details

4. Data Processing Agreements (DPA)

For all contact data submitted through this website:

  • Our hosting provider has signed a Data Processing Agreement
  • They process data only on our instructions
  • They implement appropriate security and confidentiality measures
  • They cooperate with data subject access requests
  • They delete data upon termination of service

5. Server Infrastructure

General Information:

  • This website is a static HTML site (minimal server processing)
  • Most functionality is client-side (runs in your browser)
  • No databases with personal information are queried in real-time
  • Email contact forms use secure submission protocols
Privacy Benefit:

A static website architecture means minimal data processing on servers, reducing security risks and improving privacy protection.

7. Your Data Rights & Security

Regarding data security, you have these rights:

  • Right to Access: Request a copy of the data we hold about you
  • Right to Verify Security: Ask about our security measures
  • Right to Delete Your Data: Request deletion of contact information at any time
  • Data Breach Notification: We will notify you within 72 hours of any security breach affecting your data

8. Responsible Disclosure

If you discover a security vulnerability on this website:

  • Please report it responsibly to: contact@flyingwithjoel.co.uk
  • Do NOT publicly disclose the vulnerability until we've had time to fix it
  • We appreciate your help in keeping our website secure

9. Regular Security Audits

This website undergoes regular security reviews:

  • SSL/TLS certificate validity checked monthly
  • Security headers verified quarterly
  • Hosting provider security certifications verified annually
  • Code updates applied promptly when available
